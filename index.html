TrueSight Intelligence Lab

version 0.3.0

Virtual machine lab environment for learning about TrueSight Pulse Intelligence.

Rather than having to worry about the numerous dependency differences between Windows, Mac OS X, Unix/Linux, we have developed this virtual machine that contains all the necessary software tools to complete the lab exercises.

It is imperative that you install the prerequisites on your laptop/desktop BEFORE attempting to complete these labs! See Prerequisites for details.
