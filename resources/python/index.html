Useful Python Packages

apachelog

Library for parsing Apache HTTPD log files.

http://code.google.com/p/apachelog

petl

petl is a general purpose Python package for extracting, transforming and loading tables of data.

http://petl.readthedocs.org/

Pure Python MySQL client library for accessing MySQL and MariaDB.
